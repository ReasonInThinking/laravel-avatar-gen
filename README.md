# 🎨 Avatar Generator for Laravel

This is a simple web tool that lets you create custom profile pictures using the **[ui-avatars.com](https://ui-avatars.com)** API. Just type a name, tweak the settings, and get your avatar instantly.

<img width="1234" height="1077" alt="generator" src="https://github.com/user-attachments/assets/0a0cb624-b162-471d-85d8-4a98701cdb0b" />

### 🌟 What you can do:
- **Personalize:** Create avatars based on any nickname.
- **Styling:** Choose colors (like the cool **Purple 🔮**), adjust sizes, and pick fonts.
- **Shape & Text:** Make it **Rounded**, use **Bold** text, or switch to **Uppercase**.
- **Formats:** Download or view your result in **PNG** or **SVG** format.
- **Dark Mode:** A sleek, dark interface that looks great on any screen.

### 🔗 Powered by:
All images are generated in real-time via the **UI Avatars API**. No local storage is needed, making it fast and lightweight.
